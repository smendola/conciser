🔒 Step 2: Install a non-Play-Store APK

  1. Download

    Grab the APK above. When it finishes, tap the download notification.

  2. Allow installs

    Enable installs from this browser when Android prompts you (Settings → Security → Install unknown apps).

  3. Tap Install

    Accept the prompt and wait for the success toast.

If you want you can disable "Install unknown apps" after NBJ Condenser is installed.
There is no need to launch the app after installation, that's not how this app works. Keep reading for instructions on how to use it.

🚀 Step 3: Use the app

This app works through Android's share sheet:

  1. Open YouTube

    Start any video in the YouTube app.

  2. Tap Share

    Use the share icon near the like/dislike buttons.

  3. Pick NBJ Condenser

    Find it in the share targets (probably under “More”).

    You may in fact need to find a second "More" item after the first :eyeroll:

  4. Share

    Send the video to NBJ Condenser and let it work.

Opening the app from the launcher looks nice, but it won't do anything useful on its own.
